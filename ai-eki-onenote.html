Ai · Eki

台語「要記得」× AI 記得你的一切

時間 × 地點 × AI — 在對的地方、對的時間，提醒對的事

The right reminder, at the right place, at the right time — powered by AI

受眾定位

核心受眾（20%）

  • 長照長者及照護體系 — 長者、家人、社工與照護員
  • 解決長照場景中的核心痛點，為起點與差異化定位

延伸受眾（100%）

  • 所有智慧型手機用戶 — 個人、家庭群組、小型團隊
  • 日常地點提醒、任務管理、習慣追蹤

核心痛點

長照長者場景

  • 忘記吃藥 — Missed Medication
  • 迷路走失風險 — Wandering Risk
  • 照顧者分身乏術 — Caregiver Burnout
  • 緊急狀況無法即時通知 — Delayed Emergency Alerts
  • 社工個案記錄繁瑣 — Tedious Case Management

一般使用者場景

  • 包裹到了忘記取貨 — Missed Package Pickup
  • 路過超市忘了順手買 — Forgot to Shop While Passing By
  • 忙碌中忘記預約行程 — Missed Appointments
  • 習慣養成難以持續 — Habit Consistency
  • 靠腦記太多事容易遺漏 — Mental Overload

五大核心功能

時間觸發提醒 — Time-Based Triggers

  • 設定特定時間點或時間範圍觸發推播
  • 長照應用：吃藥時間、回診前提醒
  • 一般應用：固定行程、習慣提醒

地點地理圍欄 — Location Geofencing

  • 進入範圍、離開範圍、到達特定地點均可觸發
  • 長照應用：靠近診所、離開居家圍欄
  • 一般應用：路過超市、靠近取貨門市

行為異常與習慣偵測 — Anomaly & Habit Detection

  • 學習日常行為基線，自動偵測非常規狀況
  • 長照應用：深夜離家、長時間未移動
  • 一般應用：健身打卡連續天數追蹤

AI 自動建立任務 — AI Auto Task Creation

  • 截圖上傳、語音輸入 → AI 解析 → 零手動輸入建任務
  • 長照應用：出院摘要截圖 → 自動建用藥與回診清單
  • 一般應用：蝦皮截圖 OCR → 取貨碼 + 門市 + 到期日自動填入

多角色智慧推播 — Multi-Role Smart Push

  • 依角色分發通知，支援個人、家人、社工同步接收
  • 長照應用：長者收到提醒、家人同步收到異常警示
  • 一般應用：家庭共享購物清單、團隊外勤任務分派

  • 電腦 Web 管理後台 — 完整個案資訊、AI 輔助建檔、批次任務指派、每日 AI 狀態摘要、異常標記追蹤

個人用戶、家庭群組、小型團隊

  • 手機（主要） — GPS 地點觸發、截圖與語音建任務、推播接收
  • 電腦（管理者） — 任務分派、成員狀態管理

AI 智慧任務流程

Zero Manual Entry — 截圖、語音、感測器 → AI 自動解析 → 結構化任務 → 精準觸發

步驟一：輸入來源

  • 截圖（蝦皮取貨通知 / 出院摘要）
  • 語音指令
  • GPS 感測器
  • 手動文字輸入

步驟二：AI 分析層

  • OCR 文字辨識
  • NLU 語意理解
  • 行為基線分析
  • 地點自動定位（OpenStreetMap Nominatim）

步驟三：結構化任務

  • 地點設定（地理圍欄座標）
  • 時間設定（特定時間或時間範圍）
  • 提醒內容（文字、清單）
  • 指派對象（本人、家人、社工）

步驟四：智慧觸發提醒

  • 個人推播通知（FCM）
  • 家人與團隊警示
  • 異常升級通知
  • 習慣打卡回饋

實際場景 — 長照長者

靠近診所，回診提醒

  • 觸發：GPS 偵測到診所 300m 內，且符合早上 09–11 時段
  • 行動：推播「記得帶健保卡與藥袋」
  • 地點觸發時間範圍

深夜離家，立即警示

  • 觸發：23:00 後離開居家地理圍欄範圍
  • 行動：即時通知家人，並共享 GPS 即時位置
  • 離開範圍異常偵測

出院截圖，自動建任務

  • 觸發：家人上傳出院診斷截圖
  • AI 解析：用藥清單、回診日期、衛教注意事項
  • 行動：自動建立每日服藥提醒與回診行事曆
  • OCR + AI零手動輸入

社工語音，批次指派

  • 觸發：社工說「週四新店區個案記得帶藥袋來」
  • AI 解析：時間、地區、對象條件，篩選符合個案
  • 行動：N 個個案各自收到精準提醒
  • 語音輸入批次指派

實際場景 — 一般使用者

蝦皮截圖，超商取貨提醒

  • 觸發：上傳蝦皮取貨通知截圖
  • AI 解析：OCR 自動辨識取貨碼、門市名稱、取件期限
  • 地圖自動搜尋門市位置供使用者確認
  • 行動：路過門市時推播「取貨碼：XXXXXX，期限 XX/XX」
  • OCR 辨識路過觸發

路過超市，購物清單彈出

  • 觸發：GPS 偵測到慣用超市 200m 內
  • 行動：彈出預先建立的購物清單
  • 地點觸發清單提醒

語音建任務，行程提醒

  • 觸發：說「下週二去家樂福買衛生紙」
  • AI 解析：時間（下週二）、地點（家樂福）、內容（買衛生紙）
  • 行動：自動建立提醒，無需手動填寫任何欄位
  • 語音輸入NLU 解析

路過健身房，習慣提醒

  • 觸發：週五下班路過健身房（設定週三、週五觸發）
  • 行動：推播「今天是健身日，已連續 12 天！」
  • 地點加時間習慣養成

技術架構

後端

  • Python FastAPI + PostgreSQL + SQLAlchemy
  • JWT 認證（python-jose）+ bcrypt 密碼雜湊
  • Alembic 資料庫版本管理

推播與通知

  • Firebase Cloud Messaging（FCM）

地圖與地理圍欄

  • OpenStreetMap + Leaflet.js（免費，無需 API Key）
  • Nominatim 地標搜尋與反向地理編碼
  • Haversine 公式計算距離（後端運算，無第三方費用）

AI 與 OCR

  • Claude AI — OCR 文字辨識、NLU 語意理解
  • OCR.space API — 繁體中文截圖辨識（免費方案 25,000 次/月）
  • Web Speech API — 瀏覽器原生語音輸入（免費）

前端

  • React 18 + Vite PWA
  • 支援桌機、手機、平板響應式介面

部署

  • Render.com — 後端 Web Service
  • 靜態托管（Netlify / Vercel）— 前端
